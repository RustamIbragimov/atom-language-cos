# Cache Object Script Atom Package
This is [Atom](https://atom.io/) package for [Cache Object Script](https://en.wikipedia.org/wiki/Cach%C3%A9_ObjectScript).<br>
Currently supported file types:
- cls
- mac

# Installation
1. Download or clone this repository
2. Copy everything to <code>~/.atom/packages/</code>
3. If you have your Atom editor open, then you need to reload it (enter <code> Window: Reload</code> in Command line)
